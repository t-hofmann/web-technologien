CSS

Mit CSS ist es möglich, die beabsichtigte Darstellungsweise von HTML-Seiten zu beschreiben.

Box-Modell

Jedes Element einer HTML-Seite ist eine solche Box. Wobei es zwei Untertypen gibt, nämlich "inline"- und "block"-Elemente. Inline-Elemente stehen innerhalb des Fließtextes, während Block-Elemente typischerweise durch Zeilenumbrüche eingegrenzt sind.

Das Box-Modell wird durch folgende drei Kompontenten definiert.

margin
Außenabstand (grün)
border
Umrandung (schwarz)
padding
Innenabstand (gelb)

Zusätzlich kommt hier ein div-Element in "rot" als Inhalt der Box zum Einsatz.

Selektoren

Mit Hilfe sogenannter Selektoren (engl. selector) werden diejenigen Elemente bestimmt, denen Eigenschaften zugewiesen werden sollen. Es gibt verschiedene Arten von Selektoren: 

SELEKTOR {
	Eigenschaft_A: WERT;
	Eigenschaft_B: WERT;
}

Längen

Um Längen festlegen zu können, sind in CSS verschiedene Einheiten definiert. Für die Bildschirmausgabe sind relative Einheiten empfehlenswert, - Gründe dafür sind die dynamische Anpassung an sich ändernde Größen von Schrift oder Fenster. Im Falle von bspw. Abständen im Zusammenhang mit Text eignet sich em besonders gut (vgl. anschließende Tabellen), - bei ex ist zu beachten, dass sich die Höhe von Kleinbuchstaben je nach Schriftart deutlich unterscheiden kann.

Farben

Farben werden typischerweise mit Hilfe von RGB-Farbcodes definiert. Die einzelnen Farbkanäle können dabei Werte von 0 bis 255 annehmen.

rgb
rgb([0..255], [0..255], [0..255]); bspw. rgb(100, 90, 220)
rgba
Mit Hilfe von rgba kann zusätzlich zur Farbe ein Wert fü den zusätzlichen Alphakanal festgelegt werden, damit ist es möglich eine Farben durchscheinend zu machen. Die Werte des Alphakanals werden mit einer Dezimalzahl von 0 (vollkommen transparent) bis 1.0 (nicht transparent, opaque).
#
Die Raute # ermöglicht es den RGB-Farbcode hexadezimal anzugeben, dabei sind zwei Schreibweisen möglich: #fff ist bspw. äquivalent zu #ffffff;
named
Viele Farben sind über ihre Namen ansprechbar, etwa:
white lightgrey grey black
red darkred
orange darkorange
yellow lightyellow
green lightgreen darkgreen lime
blue lightblue darkblue
purple
Beispiele
Weiß: #fff, #ffffff, rgb(255,255,255), rgba(255,255,255,0), white>
Schwarz: #000, #000000, rgb(0,0,0), rgba(0,0,0,0), black

Vertikale Ausrichtung

Mit der Eigenschaft vertical-align wird die vertikale Ausrichtung festgelegt, möglich sind die Werte top, middle und bottom. Hier gezeigt am Beispiel einer Tabelle:

vertical-align: top; vertical-align: middle; vertical-align: bottom;
