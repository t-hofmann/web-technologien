Digitale Medien

Diese Website dient als Begleitmaterial zur Vorlesung "Digitale Medien". Die dargestellten Themen sind Web-Technologien, im Speziellen HTML, CSS und Javascript. Die Darstellung folgt dem Inhalt, womit die Website selbst auch als Anwendungsbeispiel der Technologien dient.

Entstehung

Ausgehend von einem bootstrap-Layout wurden zun√§chst die Inhalte (html, css) auf einer einzelnen HTML-Seite dargestellt. Im Zuge der weiteren Ausarbeitung erschien es zweckmäßig, die Inhalte auf mehrere Unterseiten zu verteilen. Daraus folgte die Notwendigkeit das Navigationsmenü entweder mehrfach bereitzustellen, - redundant auf jeder Unterseite, - oder aber mit Hilfe von Skripting die Inhalte zu kombinieren. In Frage kämen hierzu sowohl server- als auch clientseitiges Skripting. Um die Website weiterhin per Download ohne erheblichen Installationsaufwand (etwa für einen Webserver) auf der Clientseite bereitstellen zu können, wurde clientseitigem Skripting der Vorzug gegeben.

Den Einstiegspunkt bietet die Datei index.html. Dort wird beim Start der Inhalt aus comp.home.html geladen. Bei Click (onclick) auf die oberen Menüpunkte wird die jeweilige Unterseite geladen und als Inhalt des <div class="content">-Tags gesetzt.

Das verwendete bootstrap-Layout wurde mittels eigener Stylesheets (style.css, lecture.css ) stellenweise angepasst.

Die Website wurde zuletzt getestet mit dem Browser "Mozilla Firefox - Developer Edition (version 64.0b8, 64-bit)" unter Linux.
