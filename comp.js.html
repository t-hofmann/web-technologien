Javascript

Einbinden von Javascript

Javascript kann direkt in einer HTML-Datei innerhalb von script-Tags geschrieben werden, oder durch einen Verweis auf eine externe Datei eingebunden werden, etwa: 


<script src="js/externe_datei.js"></script>
<script src="js/externe_datei.js" async></script>

Sofern die Ausführung des Skriptes asynchron erfolgen kann, sollte es mit dem HTML-Attribut "async" als asynchron ladbare Datei gekennzeichnet werden. So kann der Browser mit dem Laden und Darstellen der HTML-Seite fortfahren, während die externe Javascript-Datei heruntergeladen wird. Andernfalls müsste der Browser warten, bis die externe Javascript-Datei verfügbar ist.

Syntax

Statements

In Javascript werden Statements mit einem Semikolon (";") beendet. Mehrere Statements können mit geschweiften Klammern ("{}") zu einem Statement zusammengefasst werden.

Kommentare

Es ist möglich mehrzeilige oder einzeilige Kommentare zu schreiben

/*
	ein mehrzeiliger Kommentar
*/

// oder auch ein Kommentar ab einem doppelten "Slash" bis zum Zeilenende

Variablen

Variablen sollten mit var deklariert werden, da die Sichtbarkeit (engl. scope) sonst global ist, und daneben nicht leicht erkennbar ist, an welcher Stelle die Variable eingeführt wurde. Variablen können bei Deklaration direkt initialisiert werden. 

'use strict'; // erzwingt die explizite Deklaration einer neuen Variablen mittels "var"

var variablenNameA;
var variablenNameB = Ausdruck; // mit Initialisierung

Funktionen

Funktionen können inkl. Parametern definiert werden. Eine Funktion muss nichts zurückgeben. 

function NAME (PARAMETER){
	return VALUE;
}

Auswahl von HTML-Elementen

Um Elemente einer HTML-Seite auswählen zu können existieren verschiedene Methoden, darunter:

document.getElementById(id)Gibt das Element mit der "id" zurück.
document.getElementsByTagName(tagname)Gibt ein Array von Elementen mit dem angegebenen Tag zurück.
document.getElementsByClassName(classname)Gibt ein Array von Elementen mit der angegebenen CSS-Klasse zurück.
